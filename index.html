Overview

Goleki.js is tiny jQuery plugin that gives you flexibility to support input with autocomplete for searching local or remote data. Currently features that provide by this package is :


  • Allow you to search remote data
  • Customize template
  • Support bootstrap


Download Goleki.js

Installation

You can download manual or if you prefer using package manager you can too.

Install with bower

{% highlight bash %} bower install goleki --save {% endhighlight %}

Install with npm

{% highlight bash %} npm install goleki --save {% endhighlight %}


Then just include the following lines of code before </body> section of your HTML.

{% highlight html %} {% endhighlight %}

That's it, you done! Now checkout example below.

Basic usage

{% highlight js %} $('.basic').goleki({ source: 'https://api.github.com/search/repositories', jsonData: 'items', templates: { item: function(obj) { return ''+ obj.full_name +''; } } }); {% endhighlight %}

Custom template

{% highlight js %} $('.custom_template').goleki({ source: 'https://api.github.com/search/repositories', jsonData: 'items', templates: { loading: 'please waiting prend...', empty: 'Unfortunately there is empty', item: function(obj) { return '\n '+ '
\n '+ ' \n '+ '
\n '+ '
\n '+ '

'+ obj.full_name +'

\n '+ ' '+ obj.owner.login +'\n '+ '
\n '+ ''; } } }); {% endhighlight %}

Working with loading

{% highlight js %} $('.with_loading').goleki({ source: 'https://api.github.com/search/repositories', jsonData: 'items', templates: { item: function(obj) { return ''+ obj.full_name +''; } } }); {% endhighlight %}

Events

{% highlight js %} $('.event-example').goleki({ source: 'https://api.github.com/search/repositories', jsonData: 'items', onSelectItem: function(e, self, $el, obj) { e.preventDefault(); console.log(e, $el, obj); alert(obj.full_name +' is owned by '+ obj.owner.login); }, templates: { item: function(obj) { return ''+ obj.full_name +''; } } }); {% endhighlight %}

Options

Name Type Default Description
source string|object null Your remote url, or local object
jsonData string null Generally api has key that contains result of data, for example take a look an api github. The key for contains result data is items


if you doesn't need this, just leave empty
typeTimeout integer 800 Delay request to api when user type in input.
onSelectItem callback null Event when user select item from autocomplete.

Templates

all below property should set inside template object
item callback <a> Template for each item
loading callback string|object Template for loading text
empty callback string Template for empty text
fail callback string Template for fail contact api
